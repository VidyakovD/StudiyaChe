# Юридические документы

Эта папка — материалы для соответствия 152-ФЗ / 38-ФЗ / 54-ФЗ.
В код они НЕ компилируются и при деплое игнорируются.

## Что в папке

- [`RKN-NOTIFICATION.md`](./RKN-NOTIFICATION.md) — что вписать в уведомление в Роскомнадзор (форма на pd.rkn.gov.ru).
- [`PRIVACY-OFFICER-ORDER.md`](./PRIVACY-OFFICER-ORDER.md) — шаблон приказа о назначении ответственного за обработку ПДн (для ИП — сам ИП).
- [`PDN-REQUEST-LOG.md`](./PDN-REQUEST-LOG.md) — шаблон журнала учёта обращений субъектов ПДн.
- [`POLICY-INTERNAL.md`](./POLICY-INTERNAL.md) — внутренний документ «Политика обработки ПДн» (тот же текст, что на /privacy, но в виде утверждаемого приказом документа).

## Чек-лист действий вне кода

- [ ] Подать уведомление об обработке ПДн через [pd.rkn.gov.ru](https://pd.rkn.gov.ru/operators-registry/notification/form/) — бесплатно, ~30 минут. Без него штраф до 500 000 ₽ для ИП.
- [ ] Заполнить плейсхолдеры в [`/privacy`](../src/app/privacy/page.tsx): ИНН, ОГРНИП, почтовый адрес.
- [ ] Завести почтовый ящик `privacy@studiache.ru` (или другой) для запросов субъектов ПДн.
- [ ] Издать приказ о назначении ответственного — взять [`PRIVACY-OFFICER-ORDER.md`](./PRIVACY-OFFICER-ORDER.md), распечатать, подписать.
- [ ] Утвердить «Политику обработки ПДн» как внутренний документ — приказом ИП.
- [ ] Завести журнал обращений (пустой excel/таблица по [`PDN-REQUEST-LOG.md`](./PDN-REQUEST-LOG.md)).
- [ ] Проверить, что бэкапы БД (если уже настроены): шифрованы до загрузки, лежат в Yandex Object Storage / Yandex 360 / на собственном сервере РФ — НЕ на личном Я.Диске.
